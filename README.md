# Suspect Registry – Java

A system for managing suspects and their communications (phone calls & SMS).
Built in 3 progressive versions.

## Versions
- **v1-console** – Basic console version
- **v2-oop** – Improved OOP structure with inheritance and abstract classes
- **v3-gui** – Swing GUI with search functionality

## Concepts used
- Abstract classes & inheritance (Communication → PhoneCall, SMS)
- ArrayList, encapsulation
- Java Swing (v3)
